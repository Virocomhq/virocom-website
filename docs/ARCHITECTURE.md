# Virocom Website Architecture

## Overview
This repository contains the official website for Virocom LLC.

The website is designed to present the company, introduce FlushingConnect, and build trust with users, partners, and potential investors.

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Hosting: Replit
- Domain: virocom.co

---

## Structure
- Landing Page
- About Section
- Product Section (FlushingConnect)
- Social / Contact Links

---

## Goals
- Present Virocom professionally
- Clearly explain the product vision
- Support credibility and trust
- Prepare for future expansion

---

## Notes
This is a simple MVP website and will evolve as the platform grows.
